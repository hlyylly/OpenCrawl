Installation Guide
1. Download and unzip the project
2. Open chrome://extensions/ and enable Developer Mode
3. Click "Load unpacked" and select the extension/ folder
4. Click the extension icon, set server to: ws://39.105.206.76:9877/ws
5. (Optional) Enter your API Key to earn credits
6. Click "Save & Reconnect"
Tip: Enable "Allow in incognito" in extension details for cookie isolation
